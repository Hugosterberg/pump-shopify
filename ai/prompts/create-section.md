# Prompt — Create New Shopify Section

Create a new Shopify section for Pump Portable.

## Section

Name: `<section-name>`

Purpose:
`<describe purpose>`

## Requirements

- Create a new file in `sections/`.
- Use valid Shopify Liquid.
- Include a valid `{% schema %}` block.
- Make content editable through section settings and/or blocks.
- Use semantic HTML.
- Use mobile-first CSS.
- Avoid unnecessary JavaScript.
- Do not edit unrelated files unless necessary.

## Design Direction

Premium Scandinavian outdoor-tech.

The section should feel:

- clean
- fast
- modern
- trustworthy
- conversion-focused

## Output

After implementation, explain:

- files changed
- what settings were added
- how to add the section in Shopify theme editor
- what to test
