# Prompt — Build Premium Homepage

Build a premium homepage for Pump Portable using Shopify sections.

## Direction

The homepage should take inspiration from the current pumpportable.com brand and products, but feel much more premium, modern, faster and more conversion-focused.

## Suggested Flow

1. Announcement bar
2. Premium hero
3. Use cases
4. Featured Perfect Combo
5. Benefits
6. How it works
7. Comparison
8. Reviews/social proof
9. FAQ
10. Final CTA

## Rules

- Use Shopify sections.
- Keep content editable in theme editor.
- Do not hardcode product prices or product data when Shopify objects can be used.
- Prefer CSS over JS.
- Mobile-first.
- Do not break app blocks or layout/theme.liquid.

## Output

Make changes in small steps and explain what to test after each step.
