# Prompt — Performance Review

Review this Shopify theme for performance issues.

## Focus

- heavy JavaScript
- large images
- render-blocking assets
- layout shift
- unnecessary animations
- too many app scripts
- duplicated CSS
- mobile performance

## Rules

- Do not remove Shopify functionality.
- Do not remove app blocks without explicit approval.
- Suggest safe improvements first.
- Prefer small targeted changes.

## Output

Provide:

1. highest-impact issues
2. safe fixes
3. risky fixes that need manual approval
4. files to inspect/change
