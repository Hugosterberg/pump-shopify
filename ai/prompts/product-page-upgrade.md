# Prompt — Product Page Upgrade

Upgrade the Pump Portable product page to feel premium, clear and conversion-focused.

## Goal

Improve the product page visually and structurally without breaking Shopify product logic.

## Preserve

- product form
- variant picker
- quantity selector
- add to cart
- dynamic checkout button
- product media
- price rendering
- availability/sold-out state
- app blocks
- schema

## Desired UX

- Product imagery should feel premium.
- Buy box should be clear and sticky-friendly on mobile.
- Key benefits should be near the CTA.
- Specs should be easy to scan.
- Perfect Combo should be clearly positioned as best value.
- FAQ and trust should reduce hesitation.

## Output

Explain:

- changed files
- protected commerce logic
- what to test
