# Testing Checklist

## After Every Section Change

Test:

- Shopify preview loads
- no Liquid errors
- section can be edited in theme editor
- mobile layout works
- desktop layout works
- images do not break layout

## Product Page

Test:

- variant selection
- quantity selector
- add to cart
- dynamic checkout button
- cart drawer/page update
- price updates
- sold-out state
- product media gallery

## Cart

Test:

- quantity increase/decrease
- remove item
- checkout button
- discounts if relevant
- mobile cart UX

## Localization

Test:

- Swedish content
- English content if enabled
- currency/market selector if enabled

## Performance

Check:

- large images
- layout shift
- unnecessary JS
- mobile speed
